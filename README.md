# LinkedListRc

This repository is a small project created to learn Rust basics, especially for those coming from a C++ background. The main goal is to explore Rust's ownership, borrowing, and memory management concepts by implementing classic data structures.


## Current Progress
- [**Single Linked List**](SingleLinkedList/README.md): The first implementation is available in the `SingleLinkedList` directory. It demonstrates how to build a single linked list in Rust, focusing on idiomatic Rust patterns and comparing them to C++ approaches.
- [**Double Linked List**](DoubleLinkedList/README.md): The second implementation explores shared ownership, interior mutability, and safe pointer management in Rust.

## Purpose
This project is intended for educational purposes and experimentation. Contributions, suggestions, and questions are welcome!
